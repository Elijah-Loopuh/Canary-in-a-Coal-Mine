controls:
- wasd = move
- tab = open debug/quit menu
- esc = exit menu
- f8 = toggle fullscreen
- p to turn off music

dev notes:
- depth is used to determine where layers are drawn, larger numbers drawn on lower layers
- depth listings below:
	- dev marker: 1000, behind everything so it disappears in game
	- backgrounds: 600
	- doors: 450
	- player: 300
	- miners: 225
	- walls: 150
	- canary: 75
	- shop background: 35
	- shop objects: 18
	- in game menus/ui readouts: 9

- sprite file names start with s
- objects file names start with o
- rooms file names start with r
- sounds file names start with m

- room dimensions are
	- 1920*960 small (layer1)
	- 1920*1920 medium (layer2)
	- 3840*1920 large (layer3)

- camera dimensions are 640*400
